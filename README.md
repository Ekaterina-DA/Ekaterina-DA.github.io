🎯 ***Цель проекта.***
Проект направлен на развитие навыков использования HTML/CSS для кастомизации дашбордов и Handlebars в Apache Superset для создания адаптивных интерфейсов.

📊 ***О проекте.***
Учебный веб-сайт, созданный для практики вёрстки и подготовки к работе с визуальными компонентами в Superset. В рамках проекта реализованы:
- Базовая структура HTML-документа
- Стилизация элементов с помощью CSS
- Работа с компонентами интерфейса

🔨 ***Технологии:***
- HTML5 — базовая разметка
- CSS3 — стилизация и оформление

📝 ***Структура проекта:***
- index.html — основной файл проекта
- styles.css — стилизация компонентов
- img - папка с иконками и изображениями для проекта
